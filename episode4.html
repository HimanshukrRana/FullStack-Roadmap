4. Schema Validation with Zod

import { z } from 'zod';
const Schema = z.object({
  compatibility_score: z.number().min(0).max(100),
  strengths: z.array(z.string().min(1)),
  weaknesses: z.array(z.string().min(1)),
  suggested_openers: z.array(z.string().min(1)).max(5)
});

async function safeParse(raw) {
  const result = Schema.safeParse(JSON.parse(raw));
  if(!result.success) throw new Error('Validation failed');
  return result.data;
}
🔒 Golden Rule: Never trust AI output. Always validate before using in code.

Agent = Perceive → Reason → Act → Observe → Repeat

// Multi-Agent Pipeline
const analysis = await analyzer.run({ profile });
const bio = await improver.run({ current: profile.bio, weaknesses: analysis.weaknesses });
const openers = await generator.run({ profile: { ...profile, bio } });

⚠️ When to use agents: Multi-step workflows, tool chaining, complex reasoning. ❌ Not for simple Q&A or creative writing.
