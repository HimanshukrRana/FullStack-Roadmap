1. Quantitative Shortcuts

// Percentage/Profit-Loss
Successive change: a% then b% = a + b + (a*b)/100
CP → SP with p% profit: SP = CP × (100+p)/100

// Time-Speed-Distance
Relative speed (same dir): |v1 - v2|
Relative speed (opp dir): v1 + v2
Train crossing: Time = (L1+L2)/relative_speed

2. Logical Reasoning

graph LR A[All A are B] --> B[Some B are C] B --> C{Conclusion: Some A are C?} C -->|No| D[Invalid]

Syllogism Venn Technique: Draw circles. Test extreme cases. "All A are B" ≠ "All B are A".

Blood Relations: Use symbols: +Male, -Female, =Married, ||Sibling. Map step-by-step.

3. Data Interpretation

  1. Identify axes, units, time periods FIRST
  2. Calculate % change, ratios before answering
  3. Eliminate extreme options quickly
  4. Watch for "per capita", "compound", "base year" traps

4. Practice Framework

  • Daily: 20 MCQs timed (30 mins)
  • Weekly: Full mock test (60 mins)
  • Review: Track accuracy by topic. Drill weak areas.
  • Resources: IndiaBIX, Testbook, previous year papers
✅ Exam Strategy: Attempt sure questions first. Skip complex ones. Review marked. No negative marking? Guess intelligently.
